# Software Requirements Specification (SRS) — Trend Intelligence Hub
## Sistema de Monitoreo, NLP y Social Listening para Marketing B2B

---

## 1. Stack Tecnológico

El stack del sistema se actualiza para soportar recolección de contexto e inteligencia cualitativa:

* **Lenguaje**: Python 3.11+
* **API Framework**: FastAPI (Para la API de servicios)
* **Dashboard**: Streamlit + Plotly Express + Custom CSS (Interfaz premium con tipografía *Outfit* y estilo oscuro)
* **Base de Datos**: PostgreSQL / Supabase (Para persistencia en la nube)
* **Almacenamiento Local**: CSV estructurado para datasets enriquecidos, JSON para sistema de caché local.
* **APIs de Datos**: Google News RSS (Zero-auth), Reddit PRAW API (Comentarios), TrendsMCP API (Tendencias globales).
* **Capa NLP & LLM**:
  * *VADER Sentiment*: Scoring cuantitativo de sentimiento.
  * *Gemini 1.5 Flash*: Motor cualitativo de inteligencia de marca y resumidor de contexto.
  * *Heurísticas locales*: Sistema de fallback para clasificación de riesgo y templates si falla la API del LLM.
* **Planificador**: APScheduler para tareas periódicas automáticas de recolección y análisis.
* **Containerización**: Docker + Docker Compose.

## 3. Requerimientos No Funcionales

### Rendimiento y Escalabilidad
* **RNF-01**: El procesamiento NLP de una tendencia (recuperación de noticias, comentarios y análisis cualitativo/cuantitativo) no debe demorar más de 5 segundos.
* **RNF-02**: El sistema de caché local debe cargar instantáneamente (< 50ms) para evitar latencias de red en el dashboard.
* **RNF-03**: La interfaz de Streamlit debe responder y actualizar gráficos de Plotly en menos de 1 segundo tras la selección del usuario.
* **RNF-04**: El sistema debe estar preparado para migrar a PostgreSQL/Supabase en producción para almacenamiento persistente y búsqueda semántica con `pgvector`.

### Disponibilidad y Confiabilidad
* **RNF-05**: Si la conexión a la API de Gemini falla, el sistema debe autogenerar un análisis e ideas basadas en reglas heurísticas locales predefinidas sin lanzar errores de ejecución.
* **RNF-06**: Si el cliente de Reddit PRAW carece de credenciales válidas, el sistema debe omitir la búsqueda en Reddit de forma silenciosa y continuar recolectando desde Google News RSS.

### Seguridad y Privacidad
* **RNF-07**: Todas las API keys (`GEMINI_API_KEY`, `REDDIT_CLIENT_ID`, etc.) deben configurarse mediante variables de entorno en un archivo `.env` local, excluyéndolas del control de versiones (`.gitignore`).
* **RNF-08**: El sistema no debe almacenar datos personales identificables (PII) de los usuarios que interactúan en Reddit o las noticias analizadas.
